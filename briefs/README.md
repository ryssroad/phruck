# Briefs

Compact editorial maps before drafting.

A brief should be short enough to hand to another role and strong enough to expose whether the piece deserves to exist.

Suggested shape:

```text
Working title:
Signal:
Claim:
Mechanism:
Human layer:
Facts:
Interpretation:
Counterargument:
Uncertainties:
Sources needed:
Owner:
Reviewers:
Publish risk:
Next action:
```
