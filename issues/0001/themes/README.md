# Issue 0001 Themes

Working title:

```text
Being Human in 2026
```

This directory breaks the first issue into thematic lanes. These are not final article titles. They are editorial containers: each lane has a question, owner, likely form, failure mode, and source need.

## Theme index

- [`00-spine-half-second.md`](00-spine-half-second.md) — the hidden center: the half-second before routing.
- [`01-interface-before-intention.md`](01-interface-before-intention.md) — the gesture reaches the interface before the thought finishes.
- [`02-fatigue-defaults-refusal.md`](02-fatigue-defaults-refusal.md) — tired people, default acceptance, and refusal cost.
- [`03-memory-offload.md`](03-memory-offload.md) — recall as query, memory as service boundary.
- [`04-work-as-routing-surface.md`](04-work-as-routing-surface.md) — the worker as queue endpoint inside agentic workflows.
- [`05-scenes-and-counter-protocols.md`](05-scenes-and-counter-protocols.md) — human scenes and small practices that preserve the half-second.

## Operating rule

A theme only belongs in issue 0001 if it can answer all three:

```text
What is the mechanism?
Where is the human scene?
How does it make refusal harder or preserve refusal capacity?
```

If a theme cannot answer those, it stays as a future issue seed instead of bloating 0001.
